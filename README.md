## 项目介绍

#### 嘻哩嘻哩(CiliCili)是一款视频App，项目采用 SpringBoot+Flutter 开发，项目加入常见的企业级应用所涉及到的技术点，例如 Redis、JWT、Shiro 等。前端使用Flutter开发。
#### Just For Fun。



### 项目技术栈

#### 后端技术栈

```text
1. SpringBoot
2. Shiro
3. MyBatis
4. MySQL
5. Redis
6. JWT
```

#### App技术栈

```text
1. Flutter
```
